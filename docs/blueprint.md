# **App Name**: Chill Smoke Catalog

## Core Features:

- Mobile-First Design: Fully responsive design, optimized for mobile devices with a focus on touch-friendly interactions and vertical scrolling.
- Vape Product Catalog: Visual catalog of vape products organized by brand, showcasing available hits and flavors. Includes high-quality images with subtle dark overlays to ensure text visibility.
- Brand Showcase: Each brand is presented as a card or section with a background image, with an accordion-style dropdown to display available hits (e.g., X hits, Y hits) and corresponding flavors when tapped.
- Instagram Ordering: Clear call-to-action buttons strategically placed throughout the site, redirecting users to the Chill Smoke Instagram page to place orders. A floating Instagram icon is always visible.
- AI-Powered Flavor Recommendation Tool: Generative AI tool that suggests vape flavors based on user input preferences such as desired taste profiles, mood, or even food pairings. It uses reasoning to decide if the desired tastes align with a valid set of flavor pairings.

## Style Guidelines:

- Primary color: Neon yellow (#DFF202), inspired by a vibrant, modern urban aesthetic.
- Background color: Dark grayish-blue (#26282B), creating a nighttime street vibe.
- Accent color: Vivid reddish-orange (#F24405), to provide strong contrast and draw attention to key actions.
- Headline font: 'Space Grotesk', sans-serif for graffiti/street/bold style.
- Body font: 'Inter', sans-serif, providing a clean, simple reading experience.
- Use clear, visible Instagram icon to encourage clicks and ordering via Instagram.
- Large touch-friendly cards optimized for mobile devices.
- Subtle accordion-style animations when expanding brands or flavor options for a modern feel.
